# 🏥 Mobile Doctor – Doctor Channeling App UI/UX Design

**MObileDoctor** is a clean, modern, and intuitive mobile UI/UX design project for a doctor channeling app, built using **Figma**.  
It provides a seamless experience for patients to find doctors, schedule appointments, make payments, and even join virtual consultations.  

The design blends healthcare professionalism with user-friendly aesthetics, using a calming blue palette to establish **trust, care, and reliability**.  
Every screen is carefully crafted to guide users smoothly through their healthcare journey—from search to consultation.  

---

## 🎯 Project Overview
The goal of **MObileDoctor** is to make booking and attending doctor appointments as simple and stress-free as possible.  
Designed for patients of all ages, the app provides quick access to healthcare professionals while keeping the experience clean, accessible, and reassuring.  

### Core Objectives:
- Simplify the doctor channeling process  
- Build trust through a clean and professional visual identity  
- Create a fast, intuitive appointment booking and payment flow  
- Ensure mobile accessibility and readability for all users  

---

## 📱 Screens Designed
This UI/UX project includes **15 fully designed mobile screens**:

1. **Logo Screen** – App splash branding  
2. **Onboarding Screen 1** – Introduction to MediLink  
3. **Onboarding Screen 2** – Why choose MediLink  
4. **Onboarding Screen 3** – Quick & secure healthcare access  
5. **Login Screen** – Secure patient login  
6. **Sign Up Screen** – Easy registration  
7. **Home Screen** – Explore services and doctors  
8. **Top Doctors Screen** – Browse recommended doctors  
9. **Doctor Details Screen** – Doctor info with date/time selection  
10. **Confirm Appointment Screen** – Review selected doctor, date, and time  
11. **Payment Information Screen** – Enter payment details  
12. **Checkout Screen** – Confirm and complete payment  
13. **Appointments Screen** – View upcoming and past appointments  
14. **Meeting with Doctor Screen** – Virtual consultation interface  
15. **Profile & Edit Profile Screens** – Manage patient info, update details  
16. **Notifications Screen** – Appointment and system alerts  
17. **Search Screen** – Find doctors by specialty, name, or location

## 🎨 Design System

### 🖌️ Color Palette
| Color Use       | HEX Code  | Description                                |
|-----------------|-----------|--------------------------------------------|
| Primary Blue    | `#04709b` | Main brand color – calm and professional   |
| Dark Blue       | `#046287` | Headers, emphasis, strong contrast         |
| Sky Blue        | `#06a0de` | Buttons, highlights, key CTAs              |
| Aqua Blue       | `#30b0e4` | Secondary elements, supportive highlights  |
| Light Blue      | `#6fc8ec` | Card backgrounds, soft visuals             |
| Pastel Blue     | `#99d8f1` | Subtle backgrounds, inactive states        |
| Very Light Blue | `#e6f6fc` | Background fill, soft contrast areas       |

This calming **blue palette** is chosen to reflect **trust, safety, and healthcare reliability**.  

---

### 🔤 Typography
Typography is designed for **readability and clarity** across all screens:  

- **Primary Font:** Rounded, clean sans-serif (e.g., **Poppins, Inter, or Manrope**)  
- **Headers:** Bold and clear, ensuring easy scanning  
- **Body Text:** Regular weight for readability, accessibility-focused  
- **Font Sizes:** Optimized for mobile UI legibility  
- **Contrast:** Strong dark text on light backgrounds, ensuring accessibility  

---

## 🛠️ Design Tool
This project was built in **Figma**, using modern design practices:  

- **Component-Based Design** – Consistent UI with reusable elements  
- **Interactive Prototypes** – Simulated booking and payment flows  
- **Auto Layouts & Constraints** – Responsive adaptability across devices
